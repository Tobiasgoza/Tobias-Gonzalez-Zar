## Dataset elegido
**Ultimate Pokémon Images & Updated Pokédex Dataset (Gen I–IX)**

Elegí este dataset porque me gusta pokemon y me pareció interesante. Contiene más de 130.000 imágenes de más de 1.000 Pokémon, organizadas en carpetas según cada Pokémon. Además tiene un archivo con información del Pokédex, como el tipo, la generación, estadísticas base, altura y peso.

## ¿Qué predeciría con este dataset?

Para entrenar un modelo con este dataset podria ser de varias formas (ya que cuenta con mucha informacion), pero yo lo usaría para reconocer qué Pokémon aparece en una imagen que le muestre. O sea, el modelo recibiría una imagen y tendría que decir qué Pokémon es, por ejemplo si es Pikachu, Charizard, Bulbasaur, etc.

## ¿Qué tipo de Machine Learning usaría?

Usaría Machine Learning supervisado, porque las imágenes ya vienen etiquetadas con el nombre del Pokémon. Eso haria que el modelo pueda aprender viendo muchos ejemplos de imágenes junto con la respuesta correcta.

## ¿Qué features serían más útiles y por qué?

Las features principales vendrían de las propias imágenes. Por ejemplo:

- Los colores del Pokémon (por ejemplo, Pikachu es amarillo).
- La forma o silueta del Pokémon.
- Detalles visuales como alas, cola, cuernos o manchas.
- El tamaño o proporciones del cuerpo.

Estas características son importantes porque muchos Pokémon tienen rasgos visuales bastante únicos, lo que ayudaría al modelo a diferenciarlos entre sí y reconocerlos incluso si la imagen cambia un poco.
